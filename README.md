Бот Анализа товаров  категорий для поиска товаров для закупки @searchmp_bot. 
Бот берет с мп статс список категорий, выдает юзеру шаблон для заполнения, 
юзер отмечает категории, интересные ему, возвращает обратно и проводит анализ по критериям по умолчанию: выручка больше 1млн, цена меньше 1000р, наличие конкурентов +-30%. 
Критерии можно менять. Бот анализирует запрос и выдает в ответ таблицу со ссылками на товар, ценой и выручкой конкурента, стоимостью логистики, эквайринга и комиссии маркетплейса.
Надо:
5.1. добавить в результат анализа тренды, 
5.2. запустить продвижение, 
5.3. настроить прием платежей (сейчас перевод админу по сбп, 
5.4. настроить параллельное использование
5.5. добавить мониторинг изменения комиссий.
5.6. Сделать тоже самое для вб  и на максе
